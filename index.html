If you looking for API for find country, country code and phone code for each country for your project, below I share the API.

I use Api Builder Lab to create the api.

METHODURLDescriptionJSON body
GET 
https://app.apibuilderlab.com/api/getreq/1/all
 Use to get all country include country code & phone code
POST 
https://app.apibuilderlab.com/api/searchreq/1
 Use to search. In this case field are country_name_aka_111752597, country_code_aka_11185948, phone_code_aka_1118105. This request JSON is to search field country name. 
{
"country_name_aka_111752597": "braz"
}

Response will be: 
{
    "data": [
        {
            "id": 30,
            "created_datetime": "2022-07-26T03:30:00.000Z",
            "country_name": "Brazil",
            "country_code": "BR",
            "phone_code": "+55"
        }
    ]
}
GET 
https://app.apibuilderlab.com/api/getreq/1/setlimit/{limit}
Example:
https://app.apibuilderlab.com/api/getreq/1/setlimit/5
This use to limit 5 record.
GET 
https://app.apibuilderlab.com/api/getreq/1/paging/{start}/{limit}
Example: 
https://app.apibuilderlab.com/api/getreq/1/paging/0/5
This use to limit record with starting point. It be useful to make pagination from your application.
GET 
https://app.apibuilderlab.com/api/getreq/1/{id_record}
Example: 
https://app.apibuilderlab.com/api/getreq/1/4
This use to limit record with starting point. It be useful to make pagination from your application.
