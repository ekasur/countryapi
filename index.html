If you looking for API for find country, country code and phone code for each country for your project, below I share the API.

I use Api Builder Lab to create the api. Use below token
x-access-token on Header request:
abl-yxwfoo1lna0g8ch



To list country with paging use below API ( use method GET ).
Page value & size value on the below url can be adjust.
https://app.apibuilderlab.com/api/getreq/1/paging?page=1&size=10


To see details each country use below API ( use method GET )
https://app.apibuilderlab.com/api/getreq/1/details?id=1


To search country use below API ( use method POST )
https://app.apibuilderlab.com/api/searchreq/1
Request Body:
{ "country_name_aka_111752597":"ind" }
